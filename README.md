# webapp_cloudcomputing
# Scalable Web App (Opinionated Defaults)

Starter scaffold for a scalable web application using:
- Next.js (React) full-stack
- PostgreSQL (Prisma) for relational storage
- Redis for caching
- Docker + docker-compose for local development
- Kubernetes manifests + Terraform skeleton for infra

See `CONTRIBUTING.md` (not included) for recommended workflow: make a branch per feature, open PRs, run tests, then merge.
